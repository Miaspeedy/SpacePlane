# SpacePlane
**SpacePlane** is a space fighter game developed in Python with PySDL3.

## ScreenShot


## Control
```
W,A,S,D - to move;
J - to attack;
K - to use item;
P - to pause;
TAB - to switch language;
F11 - toggle fullscreen;
ESC - back to title
```

## Third-party libraries
* [PySDL3](https://github.com/Aermoss/PySDL3)
* [loguru](https://github.com/Delgan/loguru)
